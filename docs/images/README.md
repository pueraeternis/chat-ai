## Adding images

1. Save as PNG or WebP, width ~1200–1600px.
2. Use `snake_case` names.
3. Link from README with a repo-relative path, e.g. `docs/images/my_feature.png`.
4. Crop or blur hostnames, paths, and secrets before commit.
